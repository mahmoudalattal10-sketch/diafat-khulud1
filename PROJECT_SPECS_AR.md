# تفاصيل ومواصفات المشروع التقنية

إليك تقرير شامل عن المواصفات التقنية لمشروع "ضيافة خلود" بناءً على الكود الحالي:

## 1. الإجابة على أسئلتك المباشرة

*   **App Router أم Pages Router؟**
    *   **الإجابة:** **App Router**
    *   **الدليل:** المشروع يستخدم مجلد `src/app` (المعيار الجديد في Next.js 13/14) وتحتوي على ملفات `layout.tsx` و `page.tsx`، مما يعني أنك تستخدم أحدث وأفضل معمارية مدعومة من Next.js.

*   **Database نوعها؟**
    *   **الإجابة:** **SQLite**
    *   **الدليل:** ملف `prisma/schema.prisma` يحدد المزود: `provider = "sqlite"`.
    *   **ملاحظة:** SQLite ممتاز للمشاريع المتوسطة وبداية المشروعات (Startups) لأنه ملف واحد محمول ولا يحتاج خادم قواعد بيانات منفصل، مما يجعله سريعاً جداً للقراءة.

*   **Traffic المتوقع (قدرة التحمل)؟**
    *   **الإجابة:** خطة Cloud Startup + SQLite + Next.js (Caching) يمكنها التعامل مع **10,000 - 50,000 زائر شهرياً** بسهولة على هذا الإعداد، وقد تصل لأكثر إذا كان الاعتماد الأكبر على تصفح الفنادق (القراءة) وليس الحجوزات المتزامنة الكثيفة في نفس الثانية.
    *   استخدام `output: 'standalone'` يجعل التطبيق خفيفاً جداً.

## 2. تفاصيل المشروع الكاملة (Tech Stack)

### الواجهة الأمامية (Frontend)
*   **الإطار العمل (Framework):** Next.js 14.2 (App Router).
*   **اللغة:** TypeScript (لضمان كود خالٍ من الأخطاء).
*   **التصميم (Styling):**
    *   **Tailwind CSS:** للتنسيقات السريعة والعصرية.
    *   **Framer Motion:** للأنيميشن والحركات السلسة (مثل القوائم والكروت).
    *   **Lucide React:** للأيقونات الحديثة.
*   **إدارة الحالة (State Management):** Zustand (خفيف وسريع جداً مقارنة بـ Redux).

### الواجهة الخلفية (Backend)
*   **بيئة التشغيل:** Node.js.
*   **الخادم (Server):** Next.js Server Actions & API Routes.
*   **قاعدة البيانات (Database):** SQLite (عبر Prisma ORM).
*   **التحقق (Validation):** Zod (للتحقق من مدخلات المستخدم ومنع الأخطاء).

### خدمات الطرف الثالث (Third-Party Services)
*   **الخرائط:** Google Maps API (لأماكن الفنادق) + Leaflet (للخرائط المجانية).
*   **الدفع الإلكتروني:** PayTabs (تم تجهيز الكود للربط معها).
*   **البريد الإلكتروني:** Nodemailer (لإرسال إشعارات الحجز).
*   **إنشاء التقارير:** jsPDF (لإنشاء فواتير PDF باللغة العربية).

### البنية التحتية والنشر (Infrastructure)
*   **النظام:** Next.js Standalone Build (يقلل حجم الملفات بنسبة 80%).
*   **الهوست:** Hostinger Cloud Startup (بيئة Node.js مدارة).
*   **الحماية:** NextAuth.js (v5 Beta) لإدارة جلسات المستخدمين وحماية لوحة التحكم.

## 3. مميزات النظام الحالية
1.  **لوحة تحكم (Admin Dashboard):** لإدارة الفنادق، الحجوزات، والعملاء.
2.  **نظام حجز:** متكامل مع اختيار التاريخ والضيوف.
3.  **متعدد اللغات (RTL):** مصمم أساساً للغة العربية.
4.  **تصميم متجاوب:** يعمل بكفاءة على الموبايل والديسك توب.
